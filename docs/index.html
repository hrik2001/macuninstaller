MacUninstaller helps you completely remove a program from your macOS system by showing you which files might be related to it.


Download:

1. Download the exectuable: Download (latest)
  • Please note that, after downloading the exectuable move the exectuable where you install software locally /usr/local/bin

2. Clone the Source Code like a chad:

$ git clone https://github.com/hrik2001/macuninstaller

Usage:

→ When using the python script, just type:
$ python cli.py -p /path/to/app

→ While using using the binary distribution:
$ macuninstaller -p /path/to/app

Either of these options will output the possible files and folders related to the app.

Flags:

-p Mention The File Path
-c Put the specific paths you want to search after this
