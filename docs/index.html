MacUninstaller

MacUninstaller helps you completely remove a program from your osX system by showing you which files might be related to it. It will be out in .

Executable

Then, move it to usr/local/bin/

Use the source code

Just do: git clone https://github.com/hrik2001/macuninstaller

Usage

When using the python script, just type python cli.py -p /path/to/app

Or this if you are using using the binary distribution macuninstaller -p /path/to/app

Either of these options will output the possible files and folders related to the app.

Flags

-p Put the path to the app after this
-c Put the specific paths you want to search after this
